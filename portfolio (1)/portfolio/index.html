SYSTEM OVERVIEW

WHAT IT DOES

A modular Python pipeline that ingests any video source, enhances visibility, detects targets, tracks them through occlusion, scores threat levels in real-time, and dispatches tactical AI analysis — all without a GPU.

[DET]
YOLOv8 Detection
ONNX model via OpenCV DNN. Auto-downloaded on first run. 640px inference with NMS. CPU-only, no CUDA required.
[TRK]
ByteTrack
Two-stage cascade tracker. Stage 1 matches high-confidence detections. Stage 2 recovers occluded targets using low-confidence detections that naive trackers discard.
[KAL]
Kinematic Kalman
6-state filter per track — position, velocity, acceleration. Extrapolates trajectories through full sensor blackout. FPS-adaptive dt keeps physics correct at any frame rate.
[THR]
Threat Scoring
Per-track 0–1 threat score combining speed, zone depth, and acceleration. Dual-zone system: outer warning ring and inner critical perimeter. Auto-triggers Groq on breach.
[LLM]
Groq Tactical AI
Non-blocking background thread sends scene JSON to llama-3.3-70b. Returns structured threat level, assessment, and operator action recommendations. 8s polling + breach cooldown.
[VIS]
Visibility Enhancement
CLAHE on LAB colour space, optional gamma lift for night mode, MOG2 motion trigger, and ECC frame stabilisation for shaky footage. All toggleable at runtime.

TECHNOLOGIES

TECH STACK

Zero GPU dependency. Runs on any x86 or ARM machine with Python 3.9+.

DETECTION
YOLOv8n
ONNX via OpenCV DNN. Auto-downloaded. CPU inference at 640px.
TRACKING
ByteTrack
Two-stage Hungarian cascade. scipy linear_sum_assignment. IoU cost matrix.
STATE EST.
Kalman 6-DOF
cv2.KalmanFilter. px, py, vx, vy, ax, ay. FPS-adaptive transition matrix.
VISION
OpenCV 4.8+
MOG2, CLAHE, ECC stabilise, DNN module, drawing primitives.
LLM API
Groq API
llama-3.3-70b-versatile. Non-blocking daemon thread. JSON scene payload.
NUMERICS
NumPy + SciPy
Matrix ops, Hungarian algorithm, IoU computation.
GUI
Tkinter
Dark-themed launcher. API key manager with .env write-back.
CONFIG
python-dotenv
Loaded at import time in config.py. .env write from GUI.
LANGUAGE
Python 3.9+
Threading, dataclasses, deque, pathlib. No Conda required.
